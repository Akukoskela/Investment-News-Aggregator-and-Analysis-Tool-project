Investment-News-Aggregator-and-Analysis-Tool-project

The idea of this project is to aggregate and analyze investment news with machine learning. I want to test how machine learning can be used with investment sector and if it (or this software) can be used when doing investment decisions. The project is developed with Angular(frontend), Supabase(database) and Python(backend)
